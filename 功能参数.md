功能参数列表
==========

下面列出了所有TWAIN功能参数的描述并按字母排序。描述格式如下：

参数名称  
----------

### 描述  
参数功能描述。

### 应用程序
（可选的）应用程序信息

### 源
（可选的）源信息

### 值
类型：参数数据结构 

### 引入版本
Version x.x

### 参考
DG_AUDIO / DAT_AUDIOFILEXFER / MSG_GET


ACAP_XFERMECH
----------

### 描述
使得应用程序和源能够鉴别常见的音频转换机制。

### 应用程序
ACAP_XFERMECH当前的设置必须与应用程序的常量一致，以此来指定音频转换机制——当开始使用DF_AUDIO / DAT____AUDIOxxxxXFER/MSG______GET操作后。

### 类型  
TW_UINT16

### 取值范围
TWSX_NATIVE  
TWSX_FILE

### 版本
Version 1.8  

### 参考
DG_AUDIO / DAT_AUDIOFILEXFER / MSG_GET
